# 旅游手册生成器

> 用浏览器生成专业旅游手册 PPT，一键下载 .pptx 文件

## 功能特性

- **表单填行程** — 图形化界面，无需代码
- **AI 智能解析** — 粘贴行程文字，自动识别填表（需 DeepSeek API Key）
- **5 套主题配色** — 丝路风 / 海滨风 / 冰雪风 / 古镇风 / 商务风
- **图片上传嵌入** — 行程图片直接写入 PPT（含自动压缩至 2MB）
- **一键生成 PPT** — 浏览器本地生成，无需服务器
- **完全离线可用** — 首次加载后无需联网，无数据上传

## 快速开始

1. 双击 `travel-ppt-generator.html` 用浏览器打开
2. 在各 Tab 填写行程信息（或粘贴文字用 AI 解析）
3. 选择喜欢的主题配色
4. 点击右下角「🎯 一键生成 PPT」
5. 浏览器自动下载 .pptx 文件

## AI 解析配置

1. 点击右上角「⚙️ API设置」
2. 填入 DeepSeek API Key（自行在 [DeepSeek Platform](https://platform.deepseek.com/) 申请，有免费额度）
3. 点击「🤖 AI智能解析」，粘贴行程文本
4. AI 自动填充所有表单字段

## 生成 PPT 结构

每份 PPT 包含以下页面：

| 页码 | 内容 |
|------|------|
| 1 | 封面（主标题、副标题、天数公里数、城市路线） |
| 2 | 行程概览（城市节点动态连线图） |
| 3 ~ N+2 | 每日行程（左侧图片 + 右侧详细信息） |
| N+3 | 特色美食推荐（3列卡片，含图片） |
| N+4 | 温馨提示（图标+文字列表） |
| N+5 | 费用说明（包含/不含双栏对比） |
| N+6 | 联系我们（联系人、电话、地址、二维码） |
| N+7 | 封底 |

## 常见问题

**下载找不到？**
部分浏览器（尤其是 Chrome）会拦截自动下载。检查地址栏右侧是否有下载提示，或查看下载文件夹。

**API Key 错误？**
检查「⚙️ API设置」中的 Key 是否正确。DeepSeek Key 格式为 `sk-` 开头。确保 Key 有余额（免费额度用完需充值）。

**图片上传失败？**
图片会自动压缩至 2MB 以内。如果原图特别大，请耐心等待压缩完成。

**PPT 生成失败？**
确保至少填写了一天的行程（行程标题不能为空），然后重试。

## 手动验证清单

生成 PPT 后，用 PowerPoint 或 WPS 打开，检查以下页面：

- [ ] **封面**：主标题、副标题是否正确显示
- [ ] **行程概览**：城市节点是否按顺序连线
- [ ] **D1行程**：左侧是否有图片（或纯色背景），右侧文字是否完整
- [ ] **美食推荐**：卡片是否整齐排列，图片是否正常显示
- [ ] **费用说明**：双栏布局是否清晰
- [ ] **联系我们**：联系信息是否正确
